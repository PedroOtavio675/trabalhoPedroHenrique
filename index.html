Introdução

Linguagens de programação são sistemas formais utilizados para criar programas de computador. Elas permitem que desenvolvedores escrevam instruções compreensíveis para máquinas.

São fundamentais para o desenvolvimento de softwares, aplicativos, sistemas operacionais, inteligência artificial, websites e diversas outras tecnologias modernas.

O objetivo desta pesquisa é apresentar a evolução histórica, características e aplicações das linguagens FORTRAN, Python, Objective-C, Delphi e PHP.

FORTRAN

Histórico

O Fortran (Formula Translation) foi a primeira linguagem de programação de alto nível amplamente utilizada. Criado pela IBM e liderado por John Backus, foi lançado em 1957 para facilitar cálculos científicos e de engenharia. Ao longo dos anos recebeu diversas atualizações, permanecendo importante em pesquisas, simulações e computação de alto desempenho.

Origem

O Fortran surgiu nos Estados Unidos dentro da IBM. Seu desenvolvimento foi motivadopela necessidade de criar uma linguagem que facilitasse cálculos matemáticos complexos utilizados em pesquisas científicas, engenharia e projetos militares. Seusucesso foi tão grande que serviu de inspiração para diversas linguagens criadas posteriormente.

Paradigma

O Fortran é classificado como uma linguagem de alto nível e inicialmente seguia oparadigma procedural ou estruturado, no qual o programa é organizado emprocedimentos e funções. Com a evolução da linguagem, versões mais recentes passaram a oferecer suporte à programação orientada a objetos, permitindo maior reutilização de código e organização de projetos maiores.

Criador(es)

Seu principal criador foi John Backus, juntamente com uma equipe de engenheiros epesquisadores da IBM. Posteriormente, John Backus recebeu diversos prêmios por suacontribuição à ciência da computação.

Finalidade Principal

O Fortran foi desenvolvido para resolver problemas matemáticos e científicos de grandecomplexidade. Atualmente, é muito utilizado em áreas como engenharia civil, engenharia mecânica, meteorologia, física, química, astronomia, economia e simulaçõescomputacionais, principalmente em supercomputadores.

Sintaxe e Semântica

A sintaxe do Fortran foi criada para ser simples e próxima da linguagemmatemática. Grande parte dos comandos utiliza palavras em inglês, como PRINT, READ, IFe DO. Sua semântica é voltada principalmente para cálculos numéricos e processamento de grandes quantidades de dados científicos.


PROGRAM Media
REAL :: nota1, nota2, media
PRINT *, "Digite duas notas:"
READ *, nota1, nota2
media = (nota1 + nota2)/2
PRINT *, "Média:", media
END PROGRAM Media

Python

Histórico

O Python é uma linguagem de programação de alto nível criada por Guido van Rossum, com desenvolvimento iniciado em 1989 e primeira versão lançada em 1991. Seu objetivo era criar uma linguagem simples, legível e eficiente. Com o tempo, tornou-se uma das linguagens mais populares do mundo, sendo amplamente utilizada em inteligência artificial, ciência de dados, automação, desenvolvimento web e computação em nuvem.

Origem

O Python surgiu no Centro de Matemática e Informática (CWI), na Holanda. Foi desenvolvido como sucessor da linguagem ABC, buscando maior simplicidade e flexibilidade. Desde sua criação é um software livre, mantido por uma grande comunidade de desenvolvedores.

Paradigma

O Python é uma linguagem multiparadigma, oferecendo suporte à: Programação estruturada; Programação orientada a objetos; Programação funcional; Programação imperativa. Essa flexibilidade permite seu uso em diversos tipos de projetos.

Criador

O criador do Python é o cientista da computação holandês Guido van Rossum. Durante muitos anos, Guido foi conhecido como o Benevolent Dictator For Life (BDFL), título informal dado ao responsável pelas decisões finais sobre a evoluçãodalinguagem. Em 2018, ele deixou essa função, e atualmente o desenvolvimento do Python é coordenado pela Python Software Foundation, organização semfins lucrativosresponsável por promover e manter a linguagem

Finalidade

O Python é uma linguagem de uso geral, utilizada principalmente em: desenvolvimento web; inteligência artificial; ciência e análise de dados; automação de tarefas; desenvolvimento de jogos; computação científica; Internet das Coisas (IoT); criação de APIs e serviços em nuvem.

Sintaxe e Semântica

A sintaxe do Python é simples e organizada. Diferentemente de outras linguagens, utiliza indentação para definir blocos de código, tornando os programas mais fáceis de ler. Seus comandos são escritos em inglês e possuem significado intuitivo.

Objective-C

Histórico

O Objective-C é uma linguagem de programação criada por Brad Cox e Tom Love no início da década de 1980. Seu objetivo era combinar a eficiência da linguagem C com os conceitos da programação orientada a objetos. A linguagem ganhou destaque quando foi adotada pela empresa NeXT e, posteriormente, pela Apple para o desenvolvimento de aplicativos para macOS e iOS. Embora hoje o Swift seja a principal linguagem da Apple, o Objective-C ainda é utilizado na manutenção de sistemas e aplicativos existentes.

Origem

O Objective-C surgiu nos Estados Unidos, desenvolvido pela empresa Stepstone. Foi inspirado na linguagem Smalltalk e criado para facilitar o desenvolvimento de aplicações orientadas a objetos com alto desempenho.

Paradigma

O Objective-C é uma linguagem multiparadigma, oferecendo suporte à: Programação orientada a objetos; Programação procedural; Programação orientada a mensagens. Essa combinação proporciona flexibilidade para o desenvolvimento de aplicações complexas.

Finalidade

O Objective-C foi desenvolvido principalmente para criar aplicativos para o ecossistema da Apple, sendo utilizado em: aplicativos para iPhone e iPad; aplicativos para computadores Mac; sistemas para macOS e iOS; manutenção de aplicações legadas.

Semântica

Sua sintaxe é baseada na linguagem C, mas adiciona recursos de programação orientada a objetos. Uma de suas principais características é o envio de mensagens para objetos utilizando colchetes ([]).

Criadores

Brad Cox e Tom Love.

Delphi

Histórico

O Delphi é uma linguagem de programação e um ambiente de desenvolvimento (IDE) lançado em 1995 pela Borland. Baseado no Object Pascal, foi criado para facilitar o desenvolvimento de aplicações com interface gráfica no Windows. Sua rapidez e facilidade de uso fizeram com que fosse amplamente utilizado em sistemas comerciais, empresariais e governamentais. Atualmente, é mantido pela Embarcadero Technologies e continua sendo utilizado principalmente na manutenção de sistemas legados e no desenvolvimento de aplicações corporativas.

Origem

O Delphi foi desenvolvido nos Estados Unidos pela Borland com o objetivo de simplificar a criação de aplicações para Windows por meio da programação visual e da orientação a objetos. Seu nome foi inspirado no Oráculo de Delfos, da Grécia Antiga.

Paradigma

O Delphi é uma linguagem multiparadigma, oferecendo suporte à: Programação estruturada; Programação orientada a objetos (POO); Programação orientada a eventos. Essa combinação torna a linguagem adequada para o desenvolvimento de sistemas de diferentes portes.

Criador

O Delphi foi desenvolvido por uma equipe de engenheiros da Borland. Atualmente, seu desenvolvimento é responsabilidade da Embarcadero Technologies.

Finalidade

O Delphi é utilizado principalmente no desenvolvimento de: sistemas empresariais (ERP); sistemas financeiros; softwares para clínicas e hospitais; sistemas escolares; automação comercial; aplicações com banco de dados; aplicativos para Windows, macOS, Android e iOS.

Sintaxe e Semântica

Sua sintaxe é baseada na linguagem Pascal, utilizando as palavras begin e end para delimitar blocos de código. As variáveis devem ser declaradas antes do uso e cada instrução normalmente termina com ponto e vírgula (;).


var

  else

PHP

Histórico

O PHP é uma linguagem de programação voltada para o desenvolvimento de aplicações web. Foi criada em 1994 por Rasmus Lerdorf, inicialmente para gerenciar informações de seu site pessoal. Com o tempo, evoluiu para uma linguagem completa, recebendo melhorias importantes como o Zend Engine e suporte à programação orientada a objetos. Atualmente, continua sendo uma das linguagens mais utilizadas na criação de sites e sistemas para a internet.

Origem

O PHP surgiu no Canadá durante um projeto pessoal de Rasmus Lerdorf. Seu objetivo inicial era facilitar a criação de páginas dinâmicas, mas rapidamente passou a ser utilizado por desenvolvedores do mundo todo para aplicações web.

Paradigma

O PHP é uma linguagem multiparadigma, oferecendo suporte à: Programação estruturada; Programação orientada a objetos (POO); Programação imperativa; Programação funcional. Essa flexibilidade permite seu uso em projetos simples e também em grandes sistemas.

Criador

O principal criador do PHP foi Rasmus Lerdorf. Posteriormente, Zeev Suraski e Andi Gutmans contribuíram significativamente para a evolução da linguagem ao desenvolverem o Zend Engine, responsável pelo desempenho das versões modernas do PHP.

Finalidade

O PHP foi desenvolvido para criar páginas web dinâmicas e aplicações que interagem com bancos de dados. É amplamente utilizado em: desenvolvimento de sites; sistemas empresariais; lojas virtuais (e-commerce); blogs; plataformas de ensino; APIs; sistemas de cadastro e autenticação.

Conclusão

Cada linguagem possui características específicas e atende necessidades distintas da computação.

FORTRAN permanece importante na computação científica, Python domina áreas modernas como IA, Objective-C foi fundamental para o ecossistema Apple, Delphi continua relevante em aplicações corporativas, e PHP permanece entre as principais tecnologias web.
