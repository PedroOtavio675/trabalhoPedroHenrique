Introdução

Linguagens de programação são sistemas formais utilizados para criar programas de computador. Elas permitem que desenvolvedores escrevam instruções compreensíveis para máquinas.

São fundamentais para o desenvolvimento de softwares, aplicativos, sistemas operacionais, inteligência artificial, websites e diversas outras tecnologias modernas.

O objetivo desta pesquisa é apresentar a evolução histórica, características e aplicações das linguagens FORTRAN, Python, Objective-C, Delphi e PHP.

FORTRAN

Histórico

Considerada uma das primeiras linguagens de alto nível da história.

Origem

Desenvolvida pela IBM em 1957.

Paradigma

Imperativo, Procedural e Orientado a Objetos.

Criador(es)

Equipe liderada por John Backus.

Finalidade Principal

Computação científica e engenharia.

Sintaxe e Semântica

Estrutura voltada para cálculos matemáticos.


PROGRAM Hello
PRINT *, "Olá Mundo!"
END PROGRAM Hello

Python

Histórico

Criada em 1991.

Origem

Desenvolvida na Holanda.

Paradigma

Orientado a Objetos, Funcional e Imperativo.

Criador

Guido van Rossum.

Finalidade

Desenvolvimento geral, IA e Ciência de Dados.

Variáveis

Tipagem dinâmica.


nome = "Mundo"

print("Olá", nome)

Objective-C

Histórico

Linguagem criada nos anos 80.

Origem

Extensão da linguagem C.

Paradigma

Orientado a Objetos.

Criadores

Brad Cox e Tom Love.

Finalidade

Aplicações para macOS e iOS.

Semântica

Mensagens entre objetos.


#import <Foundation/Foundation.h>

int main() {
 NSLog(@"Olá Mundo");
 return 0;
}

Delphi

Histórico

Lançada em 1995.

Origem

Baseada em Object Pascal.

Paradigma

Orientado a Objetos.

Criador

Borland.

Finalidade

Aplicações Desktop.

Variáveis

Tipagem forte.


program Ola;

begin
 writeln('Olá Mundo');
end.

PHP

Histórico

Criada em 1995.

Origem

Desenvolvida para web.

Paradigma

Imperativo e Orientado a Objetos.

Criador

Rasmus Lerdorf.

Finalidade

Desenvolvimento Web.

Variáveis

Utilizam "$".


<?php
echo "Olá Mundo!";
?>

Comparação

Linguagem Ano Criador Paradigma Aplicação
FORTRAN 1957 John Backus Procedural Ciência
Python 1991 Guido van Rossum Multi-paradigma IA e Dados
Objective-C 1984 Brad Cox OO iOS/macOS
Delphi 1995 Borland OO Desktop
PHP 1995 Rasmus Lerdorf Imperativo/OO Web

Conclusão

Cada linguagem possui características específicas e atende necessidades distintas da computação.

FORTRAN permanece importante na computação científica, Python domina áreas modernas como IA, Objective-C foi fundamental para o ecossistema Apple, Delphi continua relevante em aplicações corporativas, e PHP permanece entre as principais tecnologias web.
